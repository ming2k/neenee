# Pursuit bar (folded into the Activity bar)

The pursuit indicator is no longer a separate footer row. The active-pursuit
objective and its `[done/total]` progress now render as a segment of the
[Activity bar](status-bar.md) (the breathing-dot row directly above the input
box), shown only while a pursuit is armed. Full pursuit detail lives inside the
**Activity modal** (click the activity bar to open it).

See [Frame layout → Footer stack](layout.md#footer-stack) for the current
footer composition.
